# Python + Flask + Docker Sample


## Overview

 python + flask + dockerでの開発環境の構築

## Requirement

Docker


## Usage

- `git clone`をしてソースコードを取得する
- プロジェクトのルートディレクトリで`docker-compose up -d`を実行する
- 環境が立ち上がれば`curl http://127.0.0.1:5000`でアクセス確認する
